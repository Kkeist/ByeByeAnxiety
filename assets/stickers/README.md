# Sticker Assets

This directory should contain 100x100 pixel PNG images for the sticker reward system.

Suggested stickers:
- star.png
- heart.png
- trophy.png
- medal.png
- flower.png
- rainbow.png
- sun.png
- moon.png
- sparkle.png
- gem.png

You can add your own custom sticker images here.
Each sticker should be:
- 100x100 pixels
- PNG format with transparency
- Named descriptively

Users will earn these stickers by completing focus sessions!
